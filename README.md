# Ceph: Учебное пособие для молодых специалистов

> **От основ школьной информатики — до промышленной эксплуатации Ceph**

## О пособии

Это учебное пособие рассчитано на администраторов и разработчиков с **базовым уровнем школьной информатики**. Версия Ceph: **Squid 19.2.x** (основная), с врезками по Tentacle 20.2.x.

### Принципы

- **От простого к сложному** — каждая глава опирается на предыдущую
- **От общего к частному** — сначала концепция, потом реализация
- **Каждый термин расшифровывается и переводится**
- **Каждая команда разбирается построчно**
- **DOT-схемы** — ортогональные стрелки, цветные, белый фон, чёрные шрифты
- **36 кейсов** — от моделирования отказа диска до полного Disaster Recovery
- **Замкнутый контур** — отдельная глава про air-gapped развёртывание
