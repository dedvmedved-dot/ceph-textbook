# Ceph: Учебное пособие для молодых специалистов

> **От основ школьной информатики — до промышленной эксплуатации Ceph**

## О пособии

Это учебное пособие рассчитано на администраторов и разработчиков, которые только начинают знакомство с распределёнными системами хранения данных. Предполагаемый уровень начальной подготовки: **базовые знания школьной информатики** (что такое файл, диск, сеть, IP-адрес).

### Принципы построения

- **От простого к сложному** — каждая глава опирается на материал предыдущей
- **От общего к частному** — сначала концепция, потом реализация в Ceph
- **Каждый термин расшифровывается** — никакого «профессионального жаргона» без объяснения
- **Каждая команда разбирается построчно** — что делает каждый флаг и параметр
- **Схемы и диаграммы** — визуализация архитектуры и потоков данных
- **Практические задания** — после каждой главы: закрепить теорию руками
- **Замкнутый контур** — отдельная глава про развёртывание без доступа в Интернет
